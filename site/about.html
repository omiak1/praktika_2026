Зачем нужен этот бот

Бот помогает узнавать новые интересные факты. Достаточно написать команду /fact и получить порцию полезной информации.

Как устроен

Бот написан на Python с использованием библиотеки python-telegram-bot. Он подключается к Telegram API и отвечает на команды пользователей.

Схема работы

Пользователь → Telegram → Bot API → Мой бот (Python) → Ответ
