# Исследование Telegram Bot API

## Как работает бот
Бот подключается к Telegram через Bot API. Когда пользователь пишет команду, Telegram отправляет запрос на сервер, бот обрабатывает и отвечает.

## Схема работы
Пользователь → Telegram → Bot API → Мой бот (Python) → Ответ

## Выбор технологий
- Python — простой язык
- python-telegram-bot — готовая библиотека
- Long polling — метод получения обновлений

## Команды бота
| Команда | Описание |
|---------|----------|
| /start | Приветствие |
| /help | Список команд |
| /fact | Случайный факт |
| /info | О боте |
